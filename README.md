# Simple Interest Calculator

This project is a Bash script that calculates simple interest.

## Formula
Simple Interest = (Principal × Rate × Time) / 100

## Usage
The script asks the user to input:
- Principal amount
- Rate of interest
- Time period

Then it calculates and displays the simple interest.

## Example
If:
Principal = 1000  
Rate = 5  
Time = 2  

Simple Interest = 100
